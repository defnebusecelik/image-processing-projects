## Use Source Code

Projeyi klonlayın

```bash
  git clone https://github.com/defnebusecelik/image-processing-projects
```

Proje dizinine gidin

```bash
  cd image-processing-projects
```

Gerekli paketleri yükleyin

```bash
  npm install mediapipe
```

Sunucuyu çalıştırın

```bash
  npm run start
```
